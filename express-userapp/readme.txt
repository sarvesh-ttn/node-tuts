User aplication made with react and node + express.
1.Client folder contains frontend logic of the app with a dedicated readme file for instructions.
2.Server folder contains the backend logic of our application with a dedicated readme file for instructions.

 Inside Server Folder

*Description:

->index.js- contains server code and is's running on port 8000.
->router folder contains code for our different requests and different http methods,inside it about.js contains code for homepage when a get request is made and users.js contains dummy data of users , on getting get/post/delete requests from home and add-user route 

*Dependencies:
->Nodemon,express

*To start 
->cd server
->on terminal write npm start.
