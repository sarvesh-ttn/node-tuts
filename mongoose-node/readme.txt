--TO_DO--
i. npm install
ii. cd node mongoose

--DESC--
Simple rest api application made with node.js and mongo db with mongoose.

 //--FOLDER STRUCTURE--//
i. routes folder contains the routing logic
ii. controller contains the server side methods used to perform operations on database
ii. model contains our databse schema in model.js as well as databse queries in service.js
ii. index.js contains the server code 

--STEPS--
npm start will start the application on port 8000.
on your terminal start mongo.

api-testing can be done on postman , end points are defined on route.js inside routes folder.
