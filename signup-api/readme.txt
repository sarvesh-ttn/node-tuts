--FOLDER STRUCTURE--
i)boot folder contains the strategy implementation and mongoose connect code.
ii)model folder contains the schema structure.
iii)router folder contains the different api endpoints logic.
iv)views folder contains ejs files to be rendered when different routes are being entered.
v) server.js contains the server code.
